# S-DOHyUNG
S-DOHyUNG is the initial of **S**imple - **D**ots and **O**ther figures **Hy**per **U**ltra **N**eatly **G**enerated.  
DOHyUNG's sounds also means "figure" in korean (도형)  
That's how this is named.
